In simple words just understand flex-wrap:wrap; property given so wrapping is occuring over cross - axis (top to bottom)

Note: We can see wrapping in column order becuase due to flex direction:column wrapping is occuring from top to bottom in several columns as we have use flex-wrap:wrap

In simple words just understand flex-wrap:wrap; property given so wrapping is occuring over cross - axis (left to right) becuase due to flex-direction : column verticall line apperaing like main-axis so for column direction cross-axis is left to right
