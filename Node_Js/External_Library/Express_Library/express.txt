const express = require("express");
const app = express(); //calling express function return object in app

• require("express") → Java ke import jaisa hai.
• express → Constructor jaisa function hai.
• express() → Java ke new Express() jaisa hai.
• app → Object hai jo Express application ko represent karta hai.
